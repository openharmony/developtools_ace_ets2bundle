## 组件 JSON schema

| 字段 | 类型 | 说明 |
|---|---|---|
| name | string | 组件名（PascalCase） |
| attrs | string[] | 属性名列表 |
| atomic | boolean | 原子组件（无子组件） |
| parents | string[] | 允许的父组件 |
| children | string[] | 允许的子组件 |
| single | boolean | 单子节点 |
| noDebugLine | boolean | 不生成 debug line |
| systemApi | boolean | 系统 API |
