# ArkUI 工具链知识库

## 概述
本知识库是 ArkUI 编译工具链的索引层，帮助开发者快速定位规格文档和开发指南。
规格文档位于 `docs/specs/`（105 篇），本知识库不重复规格内容，仅提供导航和速查。

## 文档目录

| 文档 | 内容 | 适用场景 |
|---|---|---|
| [01-架构.md](01-架构.md) | 双工具链架构、插件流水线、AST 差异 | 理解整体架构、选择目标工具链 |
| [02-装饰器速查.md](02-装饰器速查.md) | 37 个装饰器 → 规格编号映射（含 deprecated） | 修改装饰器转换或校验时定位 |
| [03-校验规则速查.md](03-校验规则速查.md) | 34 条规则 → 规格编号映射（含错误码） | 添加/修改校验规则时定位 |
| [04-组件速查.md](04-组件速查.md) | 145 个内置组件 + 7 个特殊组件处理 | 修改组件定义或转换逻辑时定位 |
| [05-常见任务导航.md](05-常见任务导航.md) | 8 类常见任务的步骤+规格引用 | 开始新任务时查阅 |

## 快速入口

- 查看某个装饰器如何转换 → [02-装饰器速查](02-装饰器速查.md)
- 查看某条校验规则的错误码 → [03-校验规则速查](03-校验规则速查.md)
- 查看某个组件是否有特殊处理 → [04-组件速查](04-组件速查.md)
- 开始一个新任务 → [05-常见任务导航](05-常见任务导航.md)
- 理解工具链整体架构 → [01-架构](01-架构.md)
- 查看工具链能力总览 → `docs/specs/README.md`
