# 功能概述
`@Prop` 装饰器标记单向同步的子组件属性，父组件值变更时同步到子组件，子组件修改不影响父组件（已 deprecated，由 `@PropRef` 替代）。

## 接口声明交叉验证

### @Prop/@PropRef 迁移发现

| 维度 | 值 |
|---|---|
| SDK 声明 | `@Prop` 已标记 `@deprecated`，由 `@PropRef` 替代 |
| 动态工具链常量 | `COMPONENT_PROP_DECORATOR = '@Prop'`（`pre_define.ts:38`）— 未做废弃标记区分 |
| 静态工具链常量（废弃） | `DeprecatedDecoratorNames.PROP = 'Prop'`（`predefines.ts`）— 通过独立枚举明确标记废弃 |
| 静态工具链常量（活跃） | `DecoratorNames.PROP_REF = 'PropRef'` |
| 工厂方法对应 | `IStateMgmtFactory.makePropRef` / `StateManagementTypes.MAKE_PROP_REF = 'makePropRef'` |

**重要发现**：`@Prop`、`@StorageProp`、`@LocalStorageProp` 已在 SDK 中标记为 `@deprecated`，被 `@PropRef`、`@StoragePropRef`、`@LocalStoragePropRef` 替代。静态工具链通过 `DeprecatedDecoratorNames` 枚举明确标记这些废弃装饰器，动态工具链仍保留原始常量但未做废弃标记区分。

## 动静态差异说明

| 维度 | 动态工具链 | 静态工具链 |
|---|---|---|
| 包装类 | `SynchedPropertySimpleOneWay`/`SynchedPropertySimpleOneWayPU` | `IPropRefDecoratedVariable<T>` 接口 |
| 工厂方法 | 直接 `new` 构造 | `STATE_MGMT_FACTORY.makePropRef` 工厂调用 |
| 复杂类型支持 | Legacy 仅支持简单类型，非简单类型报错 | Partial Update 模式支持 `SynchedPropertyObjectOneWayPU`；静态通过泛型自动处理 |
| deprecated 状态 | `@Prop` 仍可用但已废弃 | 声明文件使用 `@PropRef`，`PropRefTranslator` 标注 `@deprecated` |
