# 功能概述
将组件的链式属性调用（`.width(100).height('50%')` 等）转换为命令式的 `Component.attr(args)` 语句序列。

| 转换规则 | 说明 |
|---|---|
| 链式属性展开 | 从外向内遍历 PropertyAccessExpression，生成 `Component.attr(args)` 序列 |
| 不可变/可变属性分离 | recycle/reuse 组件优化：字面量属性放入 immutableStatements，动态属性放入 updateStatements |
| attributeModifier 特殊处理 | 生成 `Component.attr.bind(this)(args)`，因 modifier 需要 this 上下文 |
| $$ / !! 参数转换 | 属性参数中的双向绑定语法在属性转换时一并处理 |

### 关键转换逻辑
- `bindComponentAttr`（line 2340）：从外向内遍历链式 `PropertyAccessExpression`（`while` 循环 line 2368），对每个属性调用 `addComponentAttr`
- `addComponentAttr`（line 3002）：普通属性走 else 分支（line 3092-3111），调用 `createFunction(identifierNode, node, temp.arguments)` 生成 `Component.attr(args)`
- `filterRegularAttrNode`（line 3118）：判断属性参数是否全部为"常规"值（字面量、枚举、regular 变量），若是则放入 immutableStatements（仅初次渲染）
- `createFunction`（line 3872）：处理 `$$`/`!!` 参数转换（line 3878-3884）和 builder 参数转换（line 3885-3887）
- attributeModifier 特殊处理（line 3096-3099）：`isAttributeModifier = true`，生成 `Component.attr.bind(this)(args)`（line 3898-3908）

### 转换前的原始代码
```typescript
@Component
struct MyComp {
  message: string = "hello"
  count: number = 0
}
```

### 转换后的代码
```typescript
class MyComp {
  private __backing_message: string | undefined
  get message(): string | undefined { return this.__backing_message as string }
  set message(value: string) { this.__backing_message = value }
  __initializeStruct(initializers, content): void {
    this.__backing_message = {
      let __id; __id = initializers;
      __id == null ? undefined : __id.__options_has_message ? initializers!.message : "hello" as string
    }
  }
}
```

### 关键转换逻辑
- `initializeStructWithRegularProperty`（line 49）生成初始化表达式：`shouldCheckNonNull` 分支生成条件表达式 `initializers?.<name> ?? <default>`
- `getterWithRegularProperty`（line 162）和 `setterWithRegularProperty`（line 186）生成 getter/setter，getter 返回值带 `as <type>` 类型断言

## 动静态差异说明

| 维度 | 动态工具链 | 静态工具链 |
|---|---|---|
| 属性处理方式 | 链式调用展开为 `Component.attr(args)` 命令式序列 | 属性作为 struct 成员变量，生成 backing field + getter/setter |
| 属性顺序 | 从外向内遍历后 reverse，保证从内向外执行 | 无顺序概念，属性在 `__initializeStruct` 中统一初始化 |
| immutable 优化 | 有 `filterRegularAttrNode` 分离不可变/可变属性 | 无此优化 |
| attributeModifier | 生成 `bind(this)` 调用 | 无此特殊处理 |
