# 功能概述
将 `$r('app.type.name')` 资源引用转换为包含编译期解析的资源 ID、类型和模块信息的对象字面量。
## 动态
### 源码参考位置
- `compiler/src/process_ui_syntax.ts:778`（`isResource`，识别 `$r`/`$rawfile` 调用）
- `compiler/src/process_ui_syntax.ts:791`（`processResourceData`，资源数据处理）
- `compiler/src/process_ui_syntax.ts:840`（`getResourceDataNode`，从编译期资源表查 ID）
- `compiler/src/process_ui_syntax.ts:1029`（`createResourceParam`，构造资源对象）
- `compiler/src/process_ui_syntax.ts:980`（`createResourceParamWithVariable`，变量参数处理）
- `compiler/src/pre_define.ts:312-325`（`RESOURCE_TYPE` 映射表）

### 转换前的原始代码
```typescript
Text($r('app.string.hello'))
Image($r('app.media.icon'))
Text($r('app.string.hello', this.count))
```

### 转换后的代码（Legacy 和 Partial Update）
```typescript
Text({ id: 16777216, type: 10003, params: [],
      bundleName: 'com.example.app', moduleName: 'entry' })
Image({ id: 16777217, type: 20000, params: [],
      bundleName: 'com.example.app', moduleName: 'entry' })
Text({ id: 16777216, type: 10003, params: [this.count],
      bundleName: 'com.example.app', moduleName: 'entry' })
```

### 关键转换逻辑
- `isResource`（line 778-782）：识别 `$r(...)` 和 `$rawfile(...)` 调用
- `processResourceData`（line 791-828）：
  - 解析 `resourceData = ['app', 'type', 'name']`
  - `getResourceDataNode`（line 840-867）：`resourceValue = resources['app']['type']['name']`，从编译期资源表查 ID
  - `resourceType = RESOURCE_TYPE['type']`
- `createResourceParam`（line 1029-1067）：构造 `{ id, type, params, bundleName, moduleName }` 对象字面量
  - `addBundleAndModuleParam`（line 933-955）：注入 `bundleName` 和 `moduleName` 属性
    - HAR 编译时用 `__harDefaultBundleName__`/`__harDefaultModuleName__`
    - 动态时用 `__BUNDLE_NAME__`/`__MODULE_NAME__` 标识符
- `createResourceParamWithVariable`（line 980-1003）：处理变量参数，API >= 22 时用 `resourceIdName` getter 动态解析 ID
- `RESOURCE_TYPE` 映射表（pre_define.ts:312-325）：
  - color=10001, float=10002, string=10003, plural=10004, boolean=10005
  - intarray=10006, integer=10007, pattern=10008, strarray=10009
  - media=20000, rawfile=30000, symbol=40000

## 静态
### 源码参考位置
- `arkui-plugins/ui-plugins/struct-translators/factory.ts:1008`（`transformResource`）
- `arkui-plugins/ui-plugins/struct-translators/factory.ts:1056`（`processStringLiteralResourceNode`）
- `arkui-plugins/ui-plugins/struct-translators/factory.ts:1186`（`generateTransformedResourceCall`）
- `arkui-plugins/common/predefines.ts:106-112`（`Dollars` 枚举）
- `arkui-plugins/common/predefines.ts:386-399`（`RESOURCE_TYPE` 常量）

### 转换前的原始代码
```typescript
Image($r('app.media.icon'))
Text($r('app.string.hello'))
```

### 转换后的代码
```typescript
Image(_r(16777217, 20000, 'bundleName', 'moduleName', 'app', 'media', 'icon'))
Text(_r(16777216, 10003, 'bundleName', 'moduleName', 'app', 'string', 'hello'))
```

### 关键转换逻辑
- `transformResource`（line 1008-1051）：分发到 `processStringLiteralResourceNode`（line 1056）或 `generateTransformedResourceCall`（line 1186）
- `generateTransformedResourceCall` 生成 `_r(id, type, bundleName, moduleName, ...args)` 调用
  - `bundleName`/`moduleName` 通过 `isDynamicName` 判断是否使用动态 `__BUNDLE_NAME__`/`__MODULE_NAME__` 占位符
  - HAR 模块下 `resourceId = -1`（运行时解析），HAP 模块下使用预解析 ID
- 常量：`Dollars.DOLLAR_RESOURCE = '$r'`、`Dollars.TRANSFORM_DOLLAR_RESOURCE = '_r'`（predefines.ts:106-112）

## 动静态差异说明

| 维度 | 动态工具链 | 静态工具链 |
|---|---|---|
| 输出形式 | 对象字面量 `{ id, type, params, bundleName, moduleName }` | 函数调用 `_r(id, type, bundleName, moduleName, ...)` |
| 资源 ID 解析 | 编译期从资源表查 ID | HAR 模块 ID=-1 运行时解析，HAP 预解析 |
| 动态参数 | API >= 22 用 `resourceIdName` getter | `isDynamicName` 判断动态占位符 |
| 资源缓存 | 无 | `ResourceSourceCache` 缓存原始字符串供 InsightIntent 使用 |
