# 功能概述
`@Env` 装饰器标记 V2 组件的环境变量属性，从框架环境（`WritableEnvKey`/`ReadonlyEnvKey`）读取系统级配置（如颜色模式、屏幕方向等），支持运行时动态更新。

## 动态
### 源码参考位置
- `compiler/src/validate_ui_syntax.ts:2716-2748`（`checkEnvDecorator`）
- `compiler/src/pre_define.ts:99`（`COMPONENT_ENV_DECORATOR = '@Env'`）

### 转换前的原始代码
```typescript
@ComponentV2
struct MyComponent {
  @Env({ value: 'WritableEnvKey.colorMode' }) colorMode: ColorMode
  build() { Text(`${this.colorMode}`) }
}
```

### 转换后的代码（Legacy）
V2 装饰器仅支持 Partial Update 模式，无 Legacy 输出。

### 关键转换逻辑
- `checkEnvDecorator`（line 2716-2748）：校验 `@Env` 属性的类型必须为白名单类（如 `ColorMode`、`DeviceType` 等），参数格式必须匹配 `WritableEnvKey.*` 或 `ReadonlyEnvKey.*`。
- 不生成额外代码，仅校验。环境变量逻辑由运行时框架处理。

## 静态

### 转换前的原始代码
```typescript
@ComponentV2
struct MyComponent {
  @Env({ value: 'WritableEnvKey.colorMode' }) colorMode: ColorMode
  build() { Text(`${this.colorMode}`) }

```

### 转换后的代码
```typescript
class MyComponent extends CustomComponentV2 {
  private __backing_colorMode: IEnvDecoratedVariable<ColorMode>
  get colorMode(): ColorMode { return this.__backing_colorMode!.get() }

  __initializeStruct(initializers, content): void {
    this.__backing_colorMode = STATE_MGMT_FACTORY.makeEnv<ColorMode>(WritableEnvKey.colorMode, "colorMode")
  }

  static __resolveDecoratorSymbols(): void {
    const __env_colorMode = WritableEnvKey.colorMode
  }
}
```

### 深度逻辑
- `makeType = MAKE_ENV`，`hasSetter = false`（不生成 setter，环境变量只读）。
- 工厂参数：环境 key 表达式（`WritableEnvKey.colorMode`）、属性名。
- parsed 阶段额外生成 `static __resolveDecoratorSymbols(): void` 方法，用于解析环境 key 符号。
- `ENV_KEY_STRING_PATTERN` 校验参数格式，将字符串 `"WritableEnvKey.colorMode"` 转换为成员表达式，并收集导入。
- 声明文件：`@Env(value: string)`，@since 24，支持 `WritableEnvKey`/`ReadonlyEnvKey` 格式。
- `arkui-plugins/common/predefines.ts:236`（`DecoratorNames.ENV = 'Env'`）

## 动静态差异说明

| 维度 | 动态工具链 | 静态工具链 |
|---|---|---|
| 代码生成 | 不生成代码，仅校验 | backing field + getter + 工厂调用 |
| key 解析 | 运行时框架处理 | 编译期转换为成员表达式 + 导入收集 |
| setter | 无 | `hasSetter = false`，不生成 setter |
| 符号解析 | 无 | `__resolveDecoratorSymbols` 静态方法 |
| 声明接口 | 无 | `@Env(value: string)`，@since 24 |
