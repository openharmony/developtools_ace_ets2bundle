# 功能概述
将 `$$` 双向数据绑定语法（如 `TextInput({$$text: this.message})`）转换为 `{ value: varExp, changeEvent: (newValue) => { varExp = newValue } }` 对象。
## 动态
### 源码参考位置
- `compiler/src/process_component_build.ts:3200`（`isDoubleDollarToChange`，`$$` 检测）
- `compiler/src/process_component_build.ts:3254`（`classifyArgumentsNum`，`$$` 参数分类）
- `compiler/src/process_component_build.ts:3306`（`generateObjectForDollar`，`$$` 对象生成）
- `compiler/src/pre_define.ts:463-490`（`PROPERTIES_ADD_DOUBLE_DOLLAR`，支持 `$$` 的组件/属性映射）
- `compiler/src/pre_define.ts:386`（`$$_VALUE = 'value'`）
- `compiler/src/pre_define.ts:396`（`$$_CHANGE_EVENT = 'changeEvent'`）

### 转换前的原始代码
```typescript
TextInput({ $$text: this.message })
Slider({ $$value: this.sliderValue })
Toggle({ $$isOn: this.isOn })
```

### 转换后的代码（Legacy 和 Partial Update）
```typescript
TextInput.create({ text: this.message, changeEvent: (newValue) => { this.message = newValue } })
Slider.create({ value: this.sliderValue, changeEvent: (newValue) => { this.sliderValue = newValue } })
Toggle.create({ isOn: this.isOn, changeEvent: (newValue) => { this.isOn = newValue } })
```

### 关键转换逻辑
- `PROPERTIES_ADD_DOUBLE_DOLLAR`（pre_define.ts:463-490）：定义哪些组件的哪些属性支持 `$$`：
  - `TextInput` -> `text`
  - `Slider` -> `value`
  - `Toggle` -> `isOn`
  - `TextArea` -> `text`
  - `Search` -> `value`
  - `Checkbox` -> `select`
  - `Radio` -> `checked`
  - `Select` -> `selected`
  - `ListItem` -> `selected`
  - `Refresh` -> `refreshing`
  - `Navigation` -> `navBarWidth` 等
- `STYLE_ADD_DOUBLE_DOLLAR`（pre_define.ts:459-460）：`bindPopup`、`visibility`、`bindContentCover`、`bindSheet` 支持 `$$`
- `isDoubleDollarToChange`（line 3200-3208）：正则 `/^(?!\$\$\.)\$\$(.|\n)+/` 匹配 `$$xxx`
- `generateObjectForDollar`（line 3306-3320）：生成 `{ value: varExp, changeEvent: (newValue) => { varExp = newValue } }`

## 静态
### 源码参考位置
- `arkui-plugins/ui-plugins/builder-lambda-translators/bindable-factory.ts:22`（`BindableFactory`）
- `arkui-plugins/ui-plugins/builder-lambda-translators/bindable-factory.ts:43`（`generateMakeBindableCall`）
- `arkui-plugins/ui-plugins/builder-lambda-translators/utils.ts:769`（`isDoubleDollarCall`）
- `arkui-plugins/common/predefines.ts:117`（`Dollars.DOLLAR_DOLLAR = '$$'`）
- `arkui-plugins/common/predefines.ts:387`（`BindableNames.MAKE_BINDABLE = 'makeBindable'`）

### 转换前的原始代码
```typescript
Radio().checked($$(this.checked))
TextInput().text($$(this.text))
```

### 转换后的代码
```typescript
Radio().checked(makeBindable(this.checked, (value) => { this.checked = value }))
TextInput().text(makeBindable(this.text, (value) => { this.text = value }))
```

### 关键转换逻辑
- `isDoubleDollarCall`（utils.ts:769-792）：识别 `$$()` 调用，要求 callee 名为 `$$`，且 declaration 来自 `@ohos.*`/`@kit.*`/`arkui.*` 模块
- `generateMakeBindableCall`（bindable-factory.ts:43-75）：
  - 收集 `makeBindable` import
  - 生成 `makeBindable(this.x, (value) => { this.x = value })` 调用
- `updateBindableProperty`（line 80-95）：对属性值中的 `$$()` 调用做替换
- `PROPERTIES_ADD_DOUBLE_DOLLAR`（`pre_define.ts:463-490`）定义 26 个组件的 28 个属性支持 `$$`
- `STYLE_ADD_DOUBLE_DOLLAR`（`pre_define.ts:459-460`）定义 4 个样式属性支持 `$$`
- `$$this` 表示组件自身的双向绑定
- 三元表达式 `cond ? a : b` 的 `$$` 绑定会对两个分支都赋值

## 动静态差异说明

| 维度 | 动态工具链 | 静态工具链 |
|---|---|---|
| 语法形式 | `{$$prop: var}` 对象属性 | `$$(var)` 函数调用 |
| 输出形式 | `{ value: var, changeEvent: (v) => { var = v } }` | `makeBindable(var, (v) => { var = v })` |
| 支持组件 | `PROPERTIES_ADD_DOUBLE_DOLLAR` 映射 | `@ohos.*`/`@kit.*`/`arkui.*` 模块范围 |
| 适用范围 | 组件构造参数 + 部分样式属性 | 属性值 + 样式调用参数 |
