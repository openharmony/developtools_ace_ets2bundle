# 功能概述
`@Once` 装饰器修饰 `@Param` 属性，使其仅在首次初始化时接收父组件传入的值，后续父组件的更新不再同步到该属性。

## 动态
### 源码参考位置
- `compiler/src/process_struct_componentV2.ts:691`（`parseOnceDecorator`）
- `compiler/src/process_struct_componentV2.ts:96`（记录到 `structInfo.onceDecoratorSet`）
- `compiler/src/pre_define.ts:99`（`COMPONENTV2_ONCE_DECORATOR = '@Once'`）

### 转换前的原始代码
```typescript
@ComponentV2
struct MyComponent {
  @Param @Once data: string = ''
  build() { Text(this.data) }
}
```

### 转换后的代码（Legacy）
V2 装饰器仅支持 Partial Update 模式，无 Legacy 输出。

### 转换后的代码（Partial Update）
```typescript
class MyComponent extends ViewV2 {
  data: string = ''
  constructor(parent, params, __localStorage, elmtId, paramsLambda, extraInfo) {
    super(parent, elmtId, extraInfo)
    this.data = ('data' in params) ? params.data : ''
    this.finalizeConstruction()
  }
  // 注意：updateStateVars 中不包含 data 的 updateParam 调用
  initialRender() { /* 命令式 */ }
  rerender() { this.updateDirtyElements() }
}
```

### 关键转换逻辑
- `parseOnceDecorator`（line 691-693）：仅设置 `propertyDecorator.hasOnce = true`，记录到 `structInfo.onceDecoratorSet`（line 96）。
- 不生成额外代码。标记为 `@Once` 的 `@Param` 属性只初始化一次，不参与后续 `updateParam` 调用。
- 在 `traverseStructInfo` 中，`@Once` 标记的 `@Param` 仍会在构造函数中初始化，但 `updateStateVars` 中不会生成对应的 `updateParam` 语句。

## 静态
### 源码参考位置
- `arkui-plugins/ui-plugins/property-translators/once.ts:46`（`OnceTranslator`）
- `arkui-plugins/common/predefines.ts`（`DecoratorNames.ONCE = 'Once'`）

### 转换前的原始代码
```typescript
@ComponentV2
struct MyComponent {
  @Param @Once data: string = ''
  build() { Text(this.data) }
}
```

### 转换后的代码
```typescript
class MyComponent extends CustomComponentV2 {
  private __backing_data: IParamOnceDecoratedVariable<string>
  get data(): string { return this.__backing_data!.get() }
  set data(value: string) { this.__backing_data!.set(value) }

  __initializeStruct(initializers, content): void {
    this.__backing_data = STATE_MGMT_FACTORY.makeParamOnce<string>("data", init)
  }
}
```

### 深度逻辑
- `makeType = MAKE_PARAM_ONCE`，`hasUpdateStruct = false`（不生成 `__updateStruct`，父组件更新不同步），`hasSetter = true`（生成 setter，子组件内部可修改）。
- `hasResetOnReuse = true`：复用时通过 `resetOnReuseWithOnceProperty` 重新计算初始值。
- 声明文件：`@Once`，`IParamOnceDecoratedVariable<T> extends IDecoratedMutableVariable, IDecoratedV2Variable`。

- `compiler/src/pre_define.ts:105`（`COMPONENTV2_ONCE_DECORATOR = '@Once'`）

## 动静态差异说明

| 维度 | 动态工具链 | 静态工具链 |
|---|---|---|
| 更新同步 | `updateStateVars` 中不包含该属性 | `hasUpdateStruct = false`，不生成 `__updateStruct` |
| setter | 普通属性可直接赋值 | 生成 setter（`hasSetter = true`） |
| 工厂方法 | 无 | `STATE_MGMT_FACTORY.makeParamOnce` |
| 声明接口 | 无 | `IParamOnceDecoratedVariable<T>` |
