# 功能概述
`@Link` 装饰器标记双向同步的子组件属性，父组件与子组件的值双向联动。

- @Link 不允许默认值（`forbiddenSpecifyDefaultValueDecorators`，`process_component_member.ts:161`）

## 动静态差异说明

| 维度 | 动态工具链 | 静态工具链 |
|---|---|---|
| 包装类 | `SynchedPropertySimpleTwoWay`/`SynchedPropertySimpleTwoWayPU` | `ILinkDecoratedVariable<T>` 接口 |
| 工厂方法 | 直接 `new` 构造 | `STATE_MGMT_FACTORY.makeLink` 工厂调用 |
| 复杂类型 | 简单类型用 `Simple`，复杂类型用 `Object` | 泛型参数自动处理 |
| 初始化 | 从 `params` 直接取值 | 条件初始化（`initializers!.data!`） |
