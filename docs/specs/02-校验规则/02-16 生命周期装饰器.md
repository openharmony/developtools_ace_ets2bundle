# 规则
校验生命周期装饰器（@ComponentInit/@ComponentAppear 等）仅用于方法、不可有参数（除 @ComponentReuse）、不可与 @Builder/@Styles 冲突、不可为 static、@ComponentActive/@ComponentInactive 互斥。

## 错误码
- 10905369：@ComponentReuse 在 V1 中参数类型不对
- 10905370：@ComponentReuse 在 V2 中有参数
- 10905371：非 @ComponentReuse 生命周期方法有参数
- 10905383：生命周期装饰器与 @Builder/@Styles 冲突

## 核心校验规则
1. 生命周期装饰器仅用于方法
2. 非 @ComponentReuse 的生命周期方法不可有参数
3. @ComponentReuse 在 @Component 中仅允许 0 或 1 个 Record/Object 类型参数
4. @ComponentReuse 在 @ComponentV2 中不可有参数
5. 生命周期装饰器不可装饰已有生命周期方法名（aboutToAppear/onDidBuild/aboutToRecycle/aboutToReuse/aboutToDisappear）
6. @ComponentActive/@ComponentInactive 互斥
7. 生命周期方法不可为 static
8. @ComponentActive/@ComponentInactive（ERROR 级别）和 @ComponentInit/@ComponentAppear 等（WARN 级别）不可与 @Builder/@Styles 同时使用

## 示例代码
### 反例
```typescript
@ComponentV2
struct MyComp {
  @ComponentInit onInit(param: string) { }    // 有参数

  @ComponentActive
  @ComponentInactive                           // 互斥
  onActive() { }

  @ComponentAppear                             // 与 @Builder 冲突
  @Builder
  onAppear() { }

  @ComponentInit
  static onStaticInit() { }                    // static
}
```

### 正例
```typescript
@ComponentV2
struct MyComp {
  @ComponentInit onInit() { }
  @ComponentAppear onAppear() { }
  @ComponentDisappear onDisappear() { }
  @ComponentActive onActive() { }
  @ComponentInactive onInactive() { }
  @ComponentReuse onReuse() { }                // V2 中无参数
  build() { Text('hello') }
}

@Component
struct V1Comp {
  @ComponentReuse onReuse(params: Record<string, Object>) { }  // V1 中允许 Record 参数
  build() { Text('hello') }
}
```
