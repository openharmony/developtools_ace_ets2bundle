# 规则
校验组件的子组件数量限制：原子组件不可有子组件、单子节点组件只能有一个子组件、特定组件只能包含指定子组件。

## 错误码
- 10905219：特定子组件限制
- 10905220：单子组件限制
- 10905221：条件单子组件限制
- 10905222：原子组件不可有子组件
- 10905223：条件原子组件限制

## 核心校验规则
1. `AUTOMIC_COMPONENT` 集合中的组件（如 Toggle 的 Checkbox/Switch 类型）不可有子组件
2. `SINGLE_CHILD_COMPONENT` 集合中的组件（如 Toggle Button 类型）只能有一个子组件
3. `SPECIFIC_CHILD_COMPONENT` 映射中的组件只能包含指定子组件
4. 条件性限制：某些组件的子组件限制取决于属性值（如 Toggle 的 type 参数）

## 示例代码
### 反例
```typescript
// 原子组件有子组件
Toggle({ type: ToggleType.Checkbox }) {
  Text('child')               // Checkbox 类型不可有子组件
}

// 单子节点组件有多个子组件
Toggle({ type: ToggleType.Button }) {
  Text('first')
  Text('second')              // Button 类型只能有一个子组件
}
```

### 正例
```typescript
Toggle({ type: ToggleType.Checkbox })
Toggle({ type: ToggleType.Button }) {
  Text('only child')
}
```
