# 规则
校验 V2 状态管理装饰器的使用限制：@Local/@Param/@Event 不可多个同时使用、@Param 无默认值时需配合 @Require、@Local 属性和普通属性不可通过构造函数外部初始化、@ObservedV2 类必须至少有一个 @Trace 属性。

## 源码参考位置
- 动态：`compiler/src/validate_ui_syntax.ts:893`（`checkDecoratorCount`，多装饰器检测）
  - `compiler/src/validate_ui_syntax.ts:2049`（`validateStructDecorator`，V1/V2 上下文校验） 中的 V2 状态管理使用校验逻辑
- 静态：
  - `arkui-plugins/collectors/ui-collectors/validators/rules/check-componentV2-state-usage.ts:36`
  - `arkui-plugins/collectors/ui-collectors/validators/rules/check-observedV2-trace-usage-validation.ts:48`

## 适用对象
- @ComponentV2 struct 成员属性
- @ObservedV2 class 成员

## 报错信息
- 静态：
  - `The member property or method cannot be decorated by multiple built-in annotations.`（多个 @Local/@Param/@Event）
  - `When a variable decorated with '@Param' is not assigned a default value, it must also be decorated with '@Require'.`
  - `In a struct decorated with '@ComponentV2', '@Require' can only be used with '@Param' or '@BuilderParam'.`
  - `The '{{decoratorName}}' property '{{key}}' in the custom component '{{componentName}}' cannot be initialized here (forbidden to specify).`（@Local/普通属性不可外部初始化）
  - `'@{{annotationName}}' can only decorate member property.`（@Local/@Param/@Event 在函数/变量/接口上）
  - `The '@ObservedV2' annotation can only be used in 'class'.`
  - `The '@Trace' annotation can only be used in 'class'.`
  - `The '@Trace' annotation can only decorate member variables within a 'class' decorated with '@ObservedV2'.`
  - `A 'class' decorated with '@ObservedV2' must contain at least one property decorated with '@Trace'.`

## 错误码
- 无数字错误码（静态工具链使用规则名标识）

## 核心校验规则
1. @Local/@Param/@Event 不可多个同时装饰同一属性
2. @Param 无默认值时必须配合 @Require
3. @ComponentV2 中 @Require 仅配合 @Param 或 @BuilderParam
4. @Local 属性和普通属性不可通过构造函数外部初始化
5. @Local/@Param/@Event 仅用于成员属性（不可用于函数/变量/接口/类型别名）
6. @ObservedV2/@Trace 仅用于 class（不可用于 struct）
7. @Trace 仅用于 @ObservedV2 类的成员变量
8. @ObservedV2 类必须至少有一个 @Trace 属性
9. 如果 class 有 @Observed，建议改为 @ObservedV2

## 示例代码
### 反例
```typescript
@ComponentV2
struct MyComp {
  @Local @Param data: number = 0     // 多个 V2 装饰器

  @Param requiredProp: number         // @Param 无默认值但未配合 @Require

  @Require @State count: number = 0  // @Require 配合了非 @Param/@BuilderParam

  @Local localProp: number = 0       // 不可通过构造函数外部初始化
}
build: MyComp({ localProp: 1 })       // 错误：不可外部初始化 @Local

@ObservedV2
class NoTraceData {                   // 缺少 @Trace 属性
  count: number = 0
}
```

### 正例
```typescript
@ComponentV2
struct MyComp {
  @Local localData: number = 0
  @Param @Require requiredParam: number
  @Param optionalParam: number = 0
  @Event onClick: () => void = () => {}
}
build: MyComp({ requiredParam: 1, optionalParam: 2, onClick: () => {} })

@ObservedV2
class MyData {
  @Trace count: number = 0
}
```

### 静态工具链处理
静态工具链通过 `check-componentV2-state-usage.ts` 校验 5 条规则（@Local/@Param/@Event 不可多个、@Param 无默认值需配合 @Require、@Require 仅配合 @Param/@BuilderParam、@Local 不可外部初始化、仅用于属性），通过 `check-observedV2-trace-usage-validation.ts` 校验 @ObservedV2/@Trace 使用限制。支持多种 `FixSuggestion`。

## 动静态差异说明

| 维度 | 动态工具链 | 静态工具链 |
|---|---|---|
| 校验入口 | `validate_ui_syntax.ts`（多处分散校验） | `check-componentV2-state-usage.ts:36`（集中校验） |
