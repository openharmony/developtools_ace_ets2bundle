# 规则
校验 V1 状态管理装饰器（@State/@Prop/@Link 等）的属性类型不可为 @ObservedV2 装饰的类；@Observed 与 @ObservedV2 不可同时使用/互相继承。

## 适用对象
- struct 声明（V1/V2 装饰器混用检查）
- struct 成员属性（V1 装饰器属性类型为 @ObservedV2 类检查）
- class 声明（@Observed/@ObservedV2 冲突检查）

## 错误码
- 10905224：@Observed 继承 @ObservedV2
- 10905225：@ObservedV2 继承 @Observed
- 10905226：@Observed + @ObservedV2 同时使用
- 10905229：@ComponentV2 与 @Component/@Reusable/@CustomDialog 同时使用
- 10905241：@Reusable + @ReusableV2 同时使用
- 10905242：@ReusableV2 不用于 @ComponentV2
- 10905348：V1 装饰器属性类型为 @ObservedV2 类

## 核心校验规则
1. @ComponentV2 不可与 @Component/@Reusable/@CustomDialog 同时使用
2. @ReusableV2 仅用于 @ComponentV2 struct
3. @Reusable 与 @ReusableV2 不可同时使用
4. V1 装饰器（@State/@Prop/@Link/@Provide/@Consume/@ObjectLink/@StorageLink/@StorageProp/@LocalStorageLink/@LocalStorageProp）属性类型不可为 @ObservedV2 装饰的类
5. @Observed 和 @ObservedV2 不可同时用于一个类
6. @ObservedV2 class 不可继承 @Observed class，反之亦然
7. interop 模式下 V2 装饰器属性类型不可为 @Observed 类

## 示例代码
### 反例
```typescript
@ObservedV2
class MyV2Data { @Trace count: number = 0 }

@Component
struct MyComp {
  @State data: MyV2Data = new MyV2Data()   // V1 @State 不可用 @ObservedV2 类
}

@Observed
@ObservedV2                              // 不可同时使用
class MyClass { }

@ComponentV2
@Component                               // 不可混用
struct MixedComp { }
```

### 正例
```typescript
@Observed
class MyData { count: number = 0 }

@Component
struct MyComp {
  @State data: MyData = new MyData()     // V1 @State 用 @Observed 类
}

@ObservedV2
class MyV2Data { @Trace count: number = 0 }

@ComponentV2
struct MyV2Comp {
  @Local data: MyV2Data = new MyV2Data() // V2 @Local 用 @ObservedV2 类
}
```
