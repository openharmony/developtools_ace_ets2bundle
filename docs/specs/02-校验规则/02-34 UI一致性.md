# 规则
校验 UI 一致性、属性不可被调用、build 根节点限制、@WrapBuilder 校验、@CustomDialog 必须包含 CustomDialogController。

## 源码参考位置
- 动态：`compiler/src/process_component_class.ts:1390`（`validateHasControllerAndControllerCount`，@CustomDialog 必须含 Controller，错误码 10905211）
- 静态：
  - `arkui-plugins/collectors/ui-collectors/validators/rules/check-ui-consistent.ts:27`
  - `arkui-plugins/collectors/ui-collectors/validators/rules/check-attribute-no-invoke.ts:25`
  - `arkui-plugins/collectors/ui-collectors/validators/rules/check-build-root-node.ts:24`
  - `arkui-plugins/collectors/ui-collectors/validators/rules/check-wrap-builder.ts:23`
  - `arkui-plugins/collectors/ui-collectors/validators/rules/check-custom-dialog-missing-controller.ts:22`

## 适用对象
- UI 一致性检查（全局）
- 属性调用表达式
- build 方法根节点
- @WrapBuilder 使用
- @CustomDialog struct

## 报错信息
- 动态：
  - `The @CustomDialog decorated custom component must contain a property of the CustomDialogController type.`（错误码 10905211）
- 静态：
  - `The attribute '{{attributeName}}' cannot be invoked.`
  - `The 'build' method must have a single root node.`
  - `The @CustomDialog decorated custom component must contain a property of the CustomDialogController type.`
  - `@WrapBuilder can only be used with a @Builder decorated function.`

## 错误码
- 10905211：@CustomDialog 缺少 CustomDialogController 属性

## 核心校验规则
1. 属性不可作为函数调用（`attribute-no-invoke`）
2. build 方法必须有且仅有一个根节点
3. @CustomDialog struct 必须包含 `CustomDialogController` 类型属性（含 union 类型 `CustomDialogController | undefined`）
4. @WrapBuilder 只能用于 @Builder 装饰的函数
5. UI 一致性检查（全局语法一致性）

## 示例代码
### 反例
```typescript
@CustomDialog
struct MyDialog {
  // 缺少 CustomDialogController 属性
  build() { Text('dialog') }
}

@Component
struct MyComp {
  build() {
    this.someProperty()          // 属性不可被调用
    Text('a')
    Text('b')                    // build 有多个根节点
  }
}
```

### 正例
```typescript
@CustomDialog
struct MyDialog {
  controller: CustomDialogController
  build() { Text('dialog') }
}

@Component
struct MyComp {
  build() {
    Column() {
      Text('a')
      Text('b')
    }
  }
}
```
