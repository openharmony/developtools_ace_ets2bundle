# 规则
校验 `@Env` 装饰器的使用限制：只能用于 `@Component`/`@ComponentV2` struct、属性不能有默认值、类型必须是指定类型、参数格式必须正确、不可与其他状态管理装饰器组合、初始化规则限制。

## 源码参考位置
- 动态：`compiler/src/validate_ui_syntax.ts:2716`（`checkEnvDecorator`，@Env 装饰器校验） 中的 `@Env` 装饰器校验逻辑
- 静态：`arkui-plugins/collectors/ui-collectors/validators/rules/check-env-decorator.ts:76`

## 适用对象
- struct 成员属性（`@Env` 装饰的属性）
- 组件初始化调用（`@Env` 属性的初始化校验）

- 动态：

- 静态：

  - `The '@Env' property cannot be specified a default value.`
  - `The '@Env' annotation can only decorate 'WindowSizeLayoutBreakpointInfo', 'SizeInVP', 'Size', 'UIEnvWindowAvoidAreaInfoVP', 'UIEnvWindowAvoidAreaInfoPX' classes and their child classes.`
  - `Invalid parameter for '@Env'. Expected 'WritableEnvKey.<member>' or 'ReadonlyEnvKey.<member>', but got '<value>'.`
  - `Invalid parameter. State variables decorated with '@Env' of '<typeName>' can only accept <expectedKey>.`
  - `The member property or method can not be decorated by multiple built-in annotations.`
  - `The property '<propName>' cannot have multiple state management annotations.`
  - `Within structs decorated with '@Component', '@Env' can only initialize regular(non-decorated) variables.`
  - `Within structs decorated with '@ComponentV2', '<decoratorName>' can only initialize variables decorated with '@Param'.`

## 错误码
- 无数字错误码（静态工具链使用规则名标识）

## 核心校验规则
1. `@Env` 只能用于 `@Component` 或 `@ComponentV2` 装饰的 struct
2. `@Env` 装饰的属性不能有默认值
3. `@Env` 装饰的属性类型必须是 `WindowSizeLayoutBreakpointInfo`/`SizeInVP`/`Size`/`UIEnvWindowAvoidAreaInfoVP`/`UIEnvWindowAvoidAreaInfoPX` 类或其子类
4. `@Env` 的参数必须是 `WritableEnvKey.<member>` 或 `ReadonlyEnvKey.<member>` 格式的字符串字面量
5. `@Env` 的参数必须与属性类型匹配（如 `SizeInVP` 类型对应 `SystemProperties.WINDOW_SIZE`）
6. `@Env` 不可与其他状态管理装饰器（@State/@Local/@Param/@Link/@Provide 等）组合使用
7. `@Component` struct 中，`@Env` 属性只能初始化普通变量（无装饰器的变量）
8. `@ComponentV2` struct 中，`@Env` 属性只能初始化 `@Param` 装饰的变量

## 示例代码
### 反例
```typescript
@Component
struct MyComp {
  @Env('SystemProperties.WINDOW_SIZE') size: SizeInVP = new SizeInVP()  // @Env 不可有默认值
  @State @Env('SystemProperties.WINDOW_SIZE') size: SizeInVP            // 不可与 @State 组合
}

struct NoComp {
  @Env('SystemProperties.WINDOW_SIZE') size: SizeInVP                  // 非 @Component/@ComponentV2 struct
}
```

### 正例
```typescript
@Component
struct MyComp {
  regularProp: SizeInVP = new SizeInVP()
  @Env('SystemProperties.WINDOW_SIZE') size: SizeInVP
  build() {
    MyChild({ regularProp: this.size })  // @Component 中 @Env 只初始化普通变量
  }
}

@ComponentV2
struct MyV2Comp {
  @Param paramProp: SizeInVP = new SizeInVP()
  @Env('SystemProperties.WINDOW_SIZE') size: SizeInVP
  build() {
    MyChild({ paramProp: this.size })    // @ComponentV2 中 @Env 只初始化 @Param 变量
  }
}
```
