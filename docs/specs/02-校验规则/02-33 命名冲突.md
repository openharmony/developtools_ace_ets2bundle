# 规则
校验 struct 名称不可与内置组件名或内置属性名相同、同一 BlockStatement 内组件 `.id()` 属性值不可重复、成员名不可与状态管理关键字同名。

## 源码参考位置
- 动态：
  - `compiler/src/validate_ui_syntax.ts:487-496`（struct 命名冲突，错误码 10905227/10905228）
  - `compiler/src/validate_ui_syntax.ts:3467-3477`（`validateStmgmtKeywords`，关键字命名冲突）
- 静态：
  - `arkui-plugins/collectors/ui-collectors/validators/rules/check-no-same-as-built-in-attribute.ts:23`
  - `arkui-plugins/collectors/ui-collectors/validators/rules/check-no-duplicate-id.ts:29`

## 适用对象
- struct 声明（命名冲突）
- 组件 `.id()` 属性（重复 ID）
- struct 成员名（关键字冲突）

## 报错信息
- 动态：
  - `The struct '${componentName}' cannot have the same name as the built-in component '${componentName}'.`（错误码 10905227）
  - `The struct '${componentName}' cannot have the same name as the built-in attribute '${componentName}'.`（错误码 10905228）
  - `Methods, properties and accessors in structures decorated by '@Component' and '@ComponentV2' cannot have name as '${itemName}'.`（WARN）
- 静态：
  - `The struct '{{structName}}' cannot have the same name as the built-in attribute '{{builtInName}}'.`
  - `The current component id "{{id}}" is duplicate with {{path}}:{{line}}:{{index}}.`

## 错误码
- 10905227：struct 与内置组件同名
- 10905228：struct 与内置属性同名

## 核心校验规则
1. struct 名称不可与 `INNER_COMPONENT_NAMES` 集合中的内置组件名相同
2. struct 名称不可与 `BUILDIN_STYLE_NAMES` 集合中的内置属性名相同
3. 同一 BlockStatement 内组件的 `.id()` 属性值不可重复
4. @Component/@ComponentV2 struct 的成员名不可与 `stmgmtWhiteList.json` 中的状态管理关键字同名

## 示例代码
### 反例
```typescript
@Component
struct Text {              // 与内置组件 Text 同名
  build() { Text('hello') }
}

@Component
struct Width {            // 与内置属性 width 同名
  build() { Text('hello') }
}

@Component
struct MyComp {
  @State render: number = 0   // render 是状态管理关键字
  build() { Text('hello') }
}

// 重复 ID
Column() {
  Text('first').id('myId')
  Text('second').id('myId')   // 重复
}
```

### 正例
```typescript
@Component
struct MyComponent {
  @State count: number = 0
  build() { Text('hello') }
}

Column() {
  Text('first').id('firstId')
  Text('second').id('secondId')
}
```

## 静态
### 源码参考位置
- `arkui-plugins/collectors/ui-collectors/validators/rules/check-no-same-as-built-in-attribute.ts:23`
- `arkui-plugins/collectors/ui-collectors/validators/rules/check-no-duplicate-id.ts:29`
### 静态工具链处理
静态工具链通过 2 个独立规则文件校验命名冲突：struct 名称不可与内置属性名相同（check-no-same-as-built-in-attribute）、组件 .id() 不可重复（check-no-duplicate-id）。

## 动静态差异说明

| 维度 | 动态工具链 | 静态工具链 |
|---|---|---|
| struct 命名 | `validate_ui_syntax.ts:487-496`（与内置组件/属性名冲突） | `check-no-same-as-built-in-attribute.ts:23` |
| 组件 ID | `validate_ui_syntax.ts` 中检测 | `check-no-duplicate-id.ts:29` |
| 关键字命名 | `validate_ui_syntax.ts:3467`（`validateStmgmtKeywords`） | 无对应 |
