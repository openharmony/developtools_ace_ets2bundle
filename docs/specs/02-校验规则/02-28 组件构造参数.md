# 规则
校验组件构造参数：@Link/@ObjectLink 必须通过构造函数初始化、@Consume/@StorageLink 等不可通过构造函数初始化、private 属性不可通过构造函数初始化、@ObjectLink 不可用字面量初始化。

## 报错信息
- 静态：
  - `The property '{{varName}}' must be initialized through the component constructor.`（@Link/@ObjectLink 未通过构造函数初始化）
  - `'@Require' decorated '{{varName}}' must be initialized through the component constructor.`
  - `The '{{decoratorName}}' property '{{varName}}' in the custom component '{{customComponentName}}' cannot be initialized here (forbidden to specify).`（@Consume/@StorageLink/@StorageProp/@LocalStorageLink/@LocalStorageProp 不可通过构造函数初始化）
  - `Property '{{propertyName}}' is private and can not be initialized through the component constructor.`
  - `Static property '{{propertyName}}' can not be initialized through the component constructor.`
  - `The 'regular' property '{{initializerName}}' cannot be assigned to the '{{parameter}}' property '{{parameterName}}'.`
  - `The '@Observed' class object must be instantiated with the 'new' keyword; initialization with an object literal is not allowed.`
  - `The type of the parent component's state variable initializing the '@Link' variable '{{propertyName}}' must match the '@Link' variable's declared type.`

## 错误码
- 无数字错误码（静态工具链使用规则名标识）

## 核心校验规则
1. @Link/@ObjectLink 属性必须通过构造函数初始化（不可省略）
2. @Require 装饰的属性必须通过构造函数初始化
3. @Consume/@StorageLink/@StorageProp/@LocalStorageLink/@LocalStorageProp 不可通过构造函数初始化
4. private 属性不可通过构造函数初始化
5. static 属性不可通过构造函数初始化（@ComponentV2 中 static 有装饰器时；@Component 中所有 static）
6. @Link/@ObjectLink 属性不可用对象字面量初始化（必须用 `this.xxx` 成员表达式或标识符引用）
7. @ObjectLink 属性必须用 `new ClassName()` 初始化，不可用对象字面量
8. @Link 属性的父组件状态变量类型必须与 @Link 变量声明类型匹配
9. @Builder 函数仅能初始化 @BuilderParam 属性
10. @BuilderParam 属性仅能被 @Builder 函数/方法/箭头函数初始化

## 示例代码
### 反例
```typescript
@Component
struct Parent {
  @State data: MyData = new MyData()
  build() {
    Child({
      link: { count: 1 },            // @Link 不可用字面量
      objectLink: { name: 'test' },   // @ObjectLink 不可用字面量
      consume: 'value',              // @Consume 不可通过构造函数初始化
      privateProp: 1                 // private 不可通过构造函数初始化
    })
  }
}

@Component
struct Child {
  @Link link: MyData
  @ObjectLink objectLink: MyObservedData
  @Consume consume: string
  private privateProp: number = 0
  build() { Text('child') }
}
```

### 正例
```typescript
@Component
struct Parent {
  @State data: MyData = new MyData()
  build() {
    Child({
      link: this.data,              // @Link 用 this.xxx 引用
      objectLink: this.observedData // @ObjectLink 用 this.xxx 引用
    })
  }
}
```
